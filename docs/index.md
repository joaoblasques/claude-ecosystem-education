# Claude Ecosystem Education

**A free, public program to become genuinely productive across the whole Claude
ecosystem — and to prove it by learning in public.**

Most people use Claude the way they'd use a search box: one question, one answer, done.
Meanwhile the ecosystem around it has grown into something much bigger — persistent
projects and memory, interactive apps you can build in a chat, skills that encode your
standards, connectors that reach into your email and calendar, a browser that acts on
your behalf, and a full agentic harness (Claude Code) that can run loops of work while
you review. The gap between "uses Claude" and "operates Claude" is now a real skill gap.

This program closes that gap in three levels — and it holds you accountable the only
way that reliably works: **in public**.

## The deal

1. **Ship daily.** Every day of the curriculum ends with something you can show:
   an artifact, a skill, a configured loop, a repo.
2. **Log publicly.** You keep a public learning log — one short page per day, committed
   to git. The commit history is your streak. No claims, just receipts.
3. **Fork to join.** This whole site is a template. Fork it, clear the log, start your
   Day 1. See [How it works](how-it-works.md).

## The path

| Level | What | Days | You need |
|---|---|---|---|
| [**1 — The Claude Apps**](curriculum/level-1/index.md) | claude.ai done right: Projects, Memory, Artifacts, Skills, Connectors, Chrome, Excel, your first loop | 1–7 | A Claude account. No code. |
| [**2 — Claude Code & the Harness**](curriculum/level-2/index.md) | The agent loop, context economy, CLAUDE.md, skills authoring, MCP, subagents, hooks | 8–21 | A terminal and curiosity. |
| [**3 — Builder**](curriculum/level-3/index.md) | Agent SDK, publish a skill or MCP server, run a production loop, teach it back | 22–30+ | Levels 1–2. |

## Learner #0

This program is being built and completed by its own first student, in public, starting
[Day 0](log/2026-07-08.md). If a lesson is wrong, the log will show it breaking — and
the [weekly review loop](colophon.md) will fix the lesson. That's the point.
